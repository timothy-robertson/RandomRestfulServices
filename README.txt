This project is a simple restful library which allows access to multiple restful APIs in C++\CLR

Documentation can be found in Debug/Documentation.rtf, and for C# the .xml in Debug will show the documentation in Visual Studio.

Currently Supported APIs:
	BaconIpsum 	<Lorum Ipsum Generator w/ Bacon>	http://baconipsum.com/
	OpenHazards	<Earthquake Probability Service>	http://www.openhazards.com/data
	ipinfobd	<Geolocation IP Lookup Service>		http://ipinfodb.com/register.php
